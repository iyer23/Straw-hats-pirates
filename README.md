## 🧠 Problem Statement

- 97% of users never read privacy policies  
- Policies are long (~4700 words) and complex  
- No standard format for data disclosure  
- Hidden third-party data sharing  
- No alerts for policy changes  
- Dark patterns force over-sharing  

---

## 💡 Solution — Audit.ly

Audit.ly converts complex privacy policies into **simple, visual “Privacy Nutrition Labels”**.

🔎 Just input:
- App link / URL  
- APK file  
- Browser extension  

➡️ Get instant insights about:
- Data collection  
- Third-party sharing  
- Risk alerts  
- Privacy score  

---

## 🏷️ Key Features

### 📊 Privacy Nutrition Label
- Simple, visual summary of policies  
- Shows data collected, retention, encryption  

### ⚖️ DPDPA Fairness Score
- Grade (A–F) based on compliance  
- Score out of 100  

### 🚨 Real-Time Danger Alerts
- Detects risky practices like:
  - Contact scraping  
  - Continuous location tracking  
  - Financial data sharing  

### 🧒 ELI15 Mode
- “Explain Like I’m 15”  
- Converts legal text → simple English  

### 🔄 App Comparison
- Compare privacy across apps  

### 🕵️ Dark Pattern Detector
- Detects manipulative UI tricks  

### 💰 Data Value Estimator
- Shows value of your personal data  

### 🔔 Policy Change Alerts
- Notifies when apps silently update policies  

---

## ⚙️ How It Works

1. 📥 Input (URL / APK / Extension)  
2. 🔍 NLP parses policy text  
3. ⚙️ DPDPA engine analyzes compliance  
4. 🏷️ Generates privacy label  
5. 🚀 Shows alerts, scores & alternatives
